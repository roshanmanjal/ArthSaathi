# 🚀 ArthSaathi

### AI-Powered Financial Companion

ArthSaathi is an intelligent personal finance platform designed to help users manage money smarter through budgeting, financial goal planning, scam detection, financial education, and AI-driven financial insights.

Built for modern users who want a simple, accessible, and intelligent financial assistant.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/roshanmanjal/ArthSaathi.git
cd ArthSaathi
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file:

```env
VITE_GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
