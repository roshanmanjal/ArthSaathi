Project: ArthSaathi AI Demo
Description: A localized AI-first financial coach, budget advisor, and scam detector.
Architecture:
- React + Vite + TypeScript
- Framer Motion for animations
- Recharts for charts
- Zustand-style modular stores using React hooks (`useAppStore.ts` acting as the aggregator)
- LocalStorage persistence for user profile, goals, learning progress, notifications, settings, and auth.

Key Features:
- Landing page with Google/Email mock Auth flow
- Dashboard with high-level financial summary
- AI Coach (Rule-based local logic)
- Scam Shield (Rule-based scam analysis)
- Goal Planner
- Financial Health Score
- Learning modules with quizzes and XP points

Folders:
- `src/components`: UI components like Sidebar, AppLayout
- `src/pages`: Application views
- `src/store`: State management hooks
- `src/utils`: Local mock AI logic engines
- `src/data`: Static catalogs (lessons, schemes)
- `useless`: Deprecated files and mock data that were removed to make the app behave like a fresh install

This is a Demo MVP that works completely offline using simulated intelligence.
